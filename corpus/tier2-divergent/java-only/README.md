# tier2-divergent / java-only

Samples that the **Java ANTLR** engine accepts but the **TS PEG** engine
rejects (Java-accepts / TS-rejects).

This bucket is currently **empty** (placeholder). It exists so the corpus
layout documented in the repo `README.md` and in
`CorpusLoader.java`'s Javadoc (`tier2-divergent/{java-only,ts-only}`) is
consistent with the directory tree.

When a Java-only divergent sample is added, drop the `.aster` file plus its
sibling `.meta.json` here and catalogue it under **Category B: Java-only** in
`DIVERGENT-MANIFEST.md`.
